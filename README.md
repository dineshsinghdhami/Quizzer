# Quizzer Hub

![License](https://img.shields.io/badge/license-Proprietary-red)
![Python](https://img.shields.io/badge/Python-3.13%2B-blue)

**Quizzer** is an interactive web-based quiz application built using **Flask** and **SQLite**, designed to let users take quizzes, view results, and interact with a clean, fast, and responsive interface.

> **Note:** This project is **not open source**. All code, files, and designs are proprietary and may not be copied, modified, or redistributed.

---

## # Features

- Secure user authentication
- Multiple-choice quiz system
- Automatic score calculation
- User performance reports
- Admin dashboard for managing users and quizzes
- Local SQLite database
- Template-based HTML frontend
- Responsive design

## # Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Configuration:** `config.py`
